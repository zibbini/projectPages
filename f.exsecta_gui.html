A simple GUI for exploring individual nest characteristics of F. exsecta nests


This small project is an exploration into the interactive visualisation capabilities of R without using shiny. It is an extension of my dissertation and aims to make the process of monitoring individual nests easier. For example, by reducing the time needed to identify a set of nests with a certain diameter.


GUI for exploring individual nest characteristics


Methods

Data preparation was carried out using packages plyr, sp, lubridate and proj4. The final GUI was built using crosstalk, leaflet and htmltools. You can find the source code for this project here. Data was provided courtesy of Walters (2018).


References
